# Local Echo Chamber

## TL;DR
This project presents an evaluation of the “Echo Chamber” LLM jailbreaking technique, described in the paper “The Echo Chamber Multi-Turn LLM Jailbreak”, across the well known lightweight large language models **Qwen3 (8B & Coder)**, **Gemma 3 (12B)**, **Llama 4 Scout (17B) 3.2**, and **gpt-oss:20b**. 

The project is motivated by the rapidly growing demand for locally deployed LLMs and the increasing popularity of projects such as OpenClaw. Its objective is to assess how vulnerable widely used lightweight models are when exposed to adversarial interactions, and how they can be manipulated for unintended or non-intended uses.

## Introduction
The evaluation starts with a manual testing of the 'Echo Chamber' technique on the aforementioned models.
