# Local Echo Chamber

## TL;DR
This project presents an evaluation of the “Echo Chamber” LLM jailbreaking technique, described in the paper “The Echo Chamber Multi-Turn LLM Jailbreak”, across several widely used lightweight large language models.

The project is motivated by the rapidly growing demand for locally deployed LLMs and the increasing popularity of projects such as OpenClaw. Its objective is to assess how vulnerable widely used lightweight models are when exposed to adversarial interactions, and how they can be manipulated for unintended or non-intended uses.
